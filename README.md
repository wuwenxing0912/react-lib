## 开始使用

### 安装 x-ui-react

```markdown
npm i x-ui-react

yarn add x-ui-react
```

### 使用 x-ui-react

```typescript
import React from 'react';
import { Button } from 'x-ui-react';
<Button type='primary'}>Button</Button>
```
