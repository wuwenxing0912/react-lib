## 开始使用

### 安装 lib-x-ui

```markdown
npm i lib-x-ui

yarn add lib-x-ui
```

### 使用 lib-x-ui

```typescript
import React from 'react';
import { Button } from 'lib-x-ui';
<Button type='primary'}>Button</Button>
```
